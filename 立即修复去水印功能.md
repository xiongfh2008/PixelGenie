# 🚨 立即修复去水印功能

## 问题
去水印功能报错：**"No image generated in response"**

## 原因
Google Gemini 模型**不支持图像编辑**，只能理解图像。

## 解决方案（3 分钟完成）

### 方案 1: 使用 ClipDrop（最推荐）⭐

#### 1️⃣ 获取 API Key
访问: https://clipdrop.co/apis
- 注册账号（可用 Google 登录）
- 复制 API Key

#### 2️⃣ 配置
编辑 `server/.env`，添加：
```env
CLIPDROP_API_KEY=你的API密钥
```

#### 3️⃣ 重启
```bash
cd server
npm run dev
```

✅ **完成！** 去水印功能现在可以正常工作了。

### 方案 3: 使用 Remove.bg

#### 1️⃣ 获取 API Key
访问: https://www.remove.bg/api
- 注册账号
- 复制 API Key

#### 2️⃣ 配置

```env

```

#### 3️⃣ 重启

```bash
cd server

```
