## 💡 服务器窗口

您现在应该看到 **2 个 PowerShell 窗口**：

1. **后端窗口** - 显示后端服务器日志
   - 端口 3001
   - 显示 API 请求日志

2. **前端窗口** - 显示 Vite 开发服务器
   - 端口 5173 或 5174
   - 显示 HMR 更新

**不要关闭这些窗口**，它们是服务器进程。

## 🔧 如何停止服务

### 方法 1: 关闭窗口
直接关闭两个 PowerShell 窗口

### 方法 2: 使用 Ctrl+C
在每个窗口中按 `Ctrl+C` 停止服务
